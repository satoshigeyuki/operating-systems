仮想マシン, Linuxのインストール

のやり方についてゆっくり解説しています.
  • Windows 10 Hyper-V に Ubuntuを入れる
  • 注意: Windows 10, Windows 11の Home Edition だとHyper-Vの有効化が面倒なようです. やり方:
    • Windows 10 Home Edition
    • Windows 11 Home Edition
      • なお, 東京大学のライセンスではEducation editionが使えますので, OSの再インストールをしてもOKなマシンがある場合, こちらを使うという手もあります
    • Linuxに仮想マシンkvmを導入する
    • Mac OS UTM に Ubuntu を入れる
    • これが一度できるようになったら, 実マシンにLinuxをインストールすることも簡単にできるでしょう. ただし, WiFiのドライバやグラフィクスに問題がないことを確かめるため, Ubuntu Desktopのインストーラで起動することを推奨します.
    • ダウンロードしたイメージファイルをUSBメモリなどに書き込んで起動します. 書き込むツールはOSによりけりです
      • Windows 例えばこれ
      • Ubuntu 標準ツール
    • USBから起動する際, 多くのPCでは起動時にF2ボタンを押すと, firmware (BIOS) メニューが出てきて, USBからの起動を選択できるようになります
